About the Prompt Manager

The DARIA Prompt Manager allows you to:

  • Create and edit structured prompts for different agent personas
  • Track performance metrics and score prompts based on effectiveness
  • Evaluate prompt versions with a comprehensive scoring system
  • Improve prompts based on feedback and actual usage
  • Maintain version history to revert to previous versions if needed

This system is designed to help you iterate and improve on prompt engineering through systematic evaluation and feedback.
